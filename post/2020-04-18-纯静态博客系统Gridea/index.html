纯静态博客系统Gridea

2020-04-18

我到现在也体验过很多博客系统，从传统的Wordpress、Typecho,到Java博客Halo、Solo,以及纯静态博客Hexo
但是我对纯静态博客一度观点是对有的人来说很好用，对有的人难用，纯静态博客确实有它的好处，你甚至不需要服务器和域名就能能最低成本做一个自己的博客，直接托管在Github或者Coding上，因为轻量化，加载速度很快，同样也无需备案，想要加快访问速度就套个CDN，这样看纯静态博客确实很优秀，不过相对我现在使用的Typecho,它甚至连评论系统都需要外挂，不过也是因为这样你可随心所欲定制属于你个人的博客。

Gridea特性

  • 你可以使用最酷的 Markdown 语法，进行快速创作

  • 你可以给文章配上精美的封面图和在文章任意位置插入图片

  • 你可以对文章进行标签分组

  • 你可以自定义菜单，甚至可以创建外部链接菜单

  • 你可以在 Windows，MacOS 或 Linux 设备上使用此客户端

  • 你可以使用 GitHub Pages 或 Coding Pages 向世界展示，未来将支持更多平台

  • 你可以进行简单的配置，接入 Gitalk 或 DisqusJS 评论系统

  • 你可以使用中文简体或英语

  • 你可以任意使用应用内默认主题或任意第三方主题，强大的主题自定义能力

  • 你可以自定义源文件夹，利用 OneDrive、百度网盘、iCloud、Dropbox 等进行多设备同步

  • 当然 Gridea 还很年轻，有很多不足，但请相信，它会不停向前
